# 15 二进制中1的个数（11）
请实现一个函数，输入一个整数，输出该数二进制表示中 1 的个数。例如，把 9 表示成二进制是 1001，有 2 位是 1。因此，如果输入 9，则该函数输出 2。
```python
from ctypes import c_int
class Solution:
    def hammingWeight(self, n: int) -> int:
        cnt = 0
        while c_int(n).value:
            cnt += 1
            n &= n - 1
        return cnt
```
